# Welcome to the Unity UI Extensions Project

The Unity UI Extensions Project is a collection of extension scripts / effects and controls to enhance your Unity UI experience. These scripts have been gathered from many sources, combined and improved over time.

There are over 70+ extension controls / effect and other utilities in the project which are listed below:

> ### [Check out the control demos on our Tumblr page](https://unityuiextensions.tumblr.com/)
